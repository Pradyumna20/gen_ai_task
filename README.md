
A single-file Streamlit chatbot with persona selection, memory, and export functionality.

## Features
- RoastBot: witty/sarcastic roasts
- ShakespeareBot: Shakespeare-style prose
- EmojiBot: emoji-first replies
- Session memory (and optional disk persistence)
- Streamlit UI with model/temperature controls, export & clear

## Requirements
- Python 3.10+
- An OpenAI API key (set `OPENAI_API_KEY` as an environment variable)

## Quick start
1. Clone the repository.
2. Install dependencies:
   pip install -r requirements.txt
